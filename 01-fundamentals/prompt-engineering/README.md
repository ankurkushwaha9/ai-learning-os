## ğŸ”‘ Key Concepts

### The CRAFT Framework

- **C**ontext: Set the scene
- **R**ole: Define who the AI should be
- **A**ction: What you want done
- **F**ormat: How you want the output
- **T**one: The style of response

### Prompt Types

1. **Zero-shot:** Direct request, no examples
2. **Few-shot:** Include examples in prompt
3. **Chain-of-thought:** Step-by-step reasoning
4. **Role-based:** Assign a persona
