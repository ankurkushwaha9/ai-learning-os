### Blog Post Outline
```
Create a detailed outline for a blog post about [TOPIC].

Target audience: [AUDIENCE]
Goal: [educate/persuade/entertain]
Word count target: [NUMBER]

Include:
- Attention-grabbing title options (3)
- Introduction hook
- 5-7 main sections with subpoints
- Conclusion with CTA
```
